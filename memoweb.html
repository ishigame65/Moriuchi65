WEBページ制作の覚書き

WEBページはいつも知っている知識だけで適当に作ってしまっているが、そればかりだといつまでも上達しないので、 ここで今更ながら新たに知ったことを整理して少しずつ自分のモノにしていこうと思う。

HTML

class = "A B"
クラスを複数指定できる。
section
章や節を示すタグ。HTML5で追加されたタグ。sectionタグには見出しタグh1～h6を付ける。 
  
<section>
  <h2>見出し1</h2>
  <p>コンテンツ</p>
</section>
article
他から独立して単体で意味を持つ記事を示すタグ。HTML5で追加されたタグ。
表示領域指定 <meta name="viewport" content=”width=device-width,initial-scale=1″>
Google推奨レスポンシブWEBデザイン適用時に指定
width=device-width表示領域幅を端末画面幅に合わせる
initial-scale=1初期ズーム倍率を1にする
リソース先読み <link rel="preconnect" href="URL">
後のアクセスに備えて、あらかじめTCPコネクションを確立する

CSS

単位
pt1px = 0.75pt、ptを使わずpxを使えば良い
em割り当てられるフォントサイズ、1行あたりの文字数を指定する場合に使用

frグリッド中の余りスペース、CSSグリッドでスペースを均等配分する場合に使用

VSCodeでファイル比較
  1. 対象ファイルを開き、一方をアクティブにする
  2. メニュー：表示／コマンドパレット
  3. 「比較」または「Compare」と入力
  4. 「ファイル:アクティブファイルを比較しています」を選択
  5. 比較対象をクリックする
